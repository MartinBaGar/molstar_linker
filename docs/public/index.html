    Mol* Linker

    • Features
    • Supported Formats
    • Install

    Mol* Linker is a browser extension for Chrome and Firefox that automatically detects molecular structure file links on GitHub and GitLab, and injects a one-click badge to open them in the Mol* web viewer — pre-configured with your preferred representation style.

    No downloads. No local software. Click a badge, see your structure in 3D.

    Features#

    • Badge injection on GitHub & GitLab file browsers and directory trees
    • Full scene builder: per-molecule representation, color, opacity controls
    • Custom highlighting rules for chains, residues, or any selection
    • Named template library — save, share, and import JSON configurations
    • Works on private / self-hosted GitLab instances
    • Handles SPA navigation (no page reload required)
    • NEW: Offline Drag & Drop functionality for local files directly from your computer

    Supported Formats#

    .pdb, .cif, .mmcif, .bcif, .gro, .mol, .mol2, .sdf, .xyz, .ent

    Install#

    Chrome Chrome Web Store (link)
    Firefox Firefox Add-ons (link)
